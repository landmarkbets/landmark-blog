# Landmark Bets Mobile Leaderboard Ads (350x50)

Three captivating ad variations for mobile placement.

## Files

### **mobile-leaderboard-350x50.html** (Primary Version)
- Gradient orange background
- Animated pulse effect
- "Find +EV Bets in Seconds" messaging
- Live EV counter animation (+4.2% EV)
- White CTA button: "Start Free"
- **Best for:** Performance/conversion campaigns

### **mobile-leaderboard-alt1.html** (Dark/Edgy)
- Dark background with animated gradient accent bar
- "Stop Losing to the Books" messaging  
- Bold, contrarian positioning
- Orange CTA: "Get +EV"
- **Best for:** Reddit, forums, edgier audiences

### **mobile-leaderboard-alt2.html** (Clean/Data-Driven)
- White background with orange border
- Shows "+5.2% Avg ROI" stat
- "Find sharp bets instantly" messaging
- Animated arrow CTA
- **Best for:** Financial blogs, data-focused audiences

## Usage

### **HTML Embed (Recommended)**
```html
<iframe src="mobile-leaderboard-350x50.html" 
        width="350" 
        height="50" 
        frameborder="0" 
        scrolling="no">
</iframe>
```

### **Direct Link**
All ads click through to `https://landmarkbets.com`

### **Convert to PNG/JPG**
Use browser screenshot tools or:
```bash
# macOS (using Chrome)
/Applications/Google\ Chrome.app/Contents/MacOS/Google\ Chrome \
  --headless --screenshot=ad.png \
  --window-size=350,50 \
  --hide-scrollbars \
  mobile-leaderboard-350x50.html
```

## Specs

- **Size:** 350x50 pixels (standard mobile leaderboard)
- **Format:** Responsive HTML5 with CSS3 animations
- **Load time:** < 0.5s
- **File size:** ~3-4KB each
- **Brand colors:** 
  - Primary: #FF6B35
  - Secondary: #FF8559
  - Accent: #FFD700 (gold)

## A/B Testing Recommendations

1. **Week 1:** Run all three variations equally
2. **Week 2-3:** Focus on top 2 performers
3. **Week 4+:** Optimize the winner

Track CTR by variation to determine best performer.

## Customization

To change messaging, edit the HTML files:
- `.message` = Main headline text
- `.cta` = Call-to-action button text  
- `.ev-counter` = Stat display (version 1 only)

All animations are CSS-based, no JavaScript required.
